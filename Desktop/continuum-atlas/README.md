# continuum.atlas

A public, read-only atlas of systems, stories, and structure — with **Soryn Base** as the Prime Chamber.

Built with Next.js App Router, static export (`output: "export"`), dark minimal UI, and reusable components.

## Features

- **Soryn Base**: Foundational principles and core axioms
- **Systems**: Interconnected networks and complex relationships
- **Stories**: Narrative threads that give context and meaning
- **Notes**: Captured insights and fragments of understanding
- **Constitution**: Governing principles and ethical framework

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone <your-repo-url>
cd continuum-atlas
```

1. Install dependencies

```bash
npm install
# or
yarn install
```

1. Run the development server

```bash
npm run dev
# or
yarn dev
```

1. Open [http://localhost:3000](http://localhost:3000) in your browser
